# 📚 My Library App

This is a simple **JavaScript-based Library App** that allows users to:

- View a preloaded list of books
- Add new books via a form dialog
- Toggle a book’s "read" status
- Delete individual books or the most recently added book

## 🛠️ Features

- ✅ Preloaded with 10 classic books
- ➕ Add new books via a modal form
- 🔄 Mark books as read/unread
- ❌ Remove individual books
- 🗑️ Remove the last book with a single click

## 💻 Technologies Used

- **HTML** – For basic structure and dialog elements
- **CSS** – For styling cards and toggle switches
- **JavaScript** – For dynamically creating, updating, and deleting book cards
